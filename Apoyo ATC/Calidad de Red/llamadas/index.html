Contenido de Preguntas Filtro

Las siguientes preguntas ayudarán a identificar la consulta del cliente. Si el cliente no llegase a entender alguna de las preguntas que realizaste, puedes reformularlas.

1. Detalle del inconveniente: no realiza, no recibe o ambos.

2. ¿Cuál es el mensaje o locución de error?

3. ¿Cuándo inició el inconveniente?

4. ¿Sucede con varios números o uno en específico?

5. ¿El problema es con un solo operador o con todos?

6. ¿Su número lo usó con otro operador antes de pertenecer a Bitel?

7. ¿Desde que realizó la portabilidad a Bitel no recibe ni realiza llamadas?

8. ¿Anteriormente ya ha realizado llamadas o las ha recibido?

Recuerda validar datos de 1er o 2do nivel después de realizar las preguntas filtro y que las preguntas para identificar la consulta solo se generarán si el cliente no deja en claro el problema. Si el cliente es específico con su consulta, pasaremos a realizar los descartes en el sistema.

Problemas con Llamadas

No Recibe Llamadas

Estimado realizar la siguiente marcación: Desvío de llamada ##002# + tecla de llamar y si tienes inconvenientes para recibir llamadas: #35*000# + tecla de llamar.

No Realiza Llamadas

Estimado cliente, realizar la siguiente marcación si tienes inconvenientes para llamar: #33*0000# + tecla llamar.
